Polarisについて

「情報が多すぎて動けない…」

「他人と比べて落ち込んでしまう…」

「自分がどこに向かっているのか分からない…」

そんな経験はありませんか？

Polarisはそんな悩みを優しく照らすロードマップ管理アプリです。

3つのメリット

  • 削る
    自分の理想をもとに選んだロードマップ→ステップ→今日のToDoに集中できるから、迷わず前に進めます。
  • 支える
    信頼できるパートナーと支え合うことで、困難な道のりも乗り越えられます。
  • 積み上げる
    ToDoを完了させ、ステップを達成し、ロードマップを完走することで、目指す場所へたどり着けます。

情報過多で疲弊していた毎日から、
スマホの中にあなただけの輝く星がある毎日へ。

夜空の北極星が旅人の道しるべとなるように。
Polarisが、あなたの進むべき道を照らします。

2つのモード
  • Growthモード: 自分の成長を管理
  • Supportモード: パートナーの成長を管理

設定画面からモードを切り替えることができます。

Polaris画面（Growth）

Polarisに選択したロードマップ・ステップの今日のToDoを表示します。

できること

  • • ToDo作成・編集・完了
  • • 右スワイプで削除・複製・繰り返しToDo作成
  • • 長押しで並べ替え
ToDo画面（Growth）

Polarisに選択したロードマップの全ステップのToDoを確認できます。

できること

  • • ToDo作成・編集・完了
  • • 右スワイプで削除・複製・繰り返しToDo作成
  • • 長押しで並べ替え

インボックスとは

ロードマップには紐付いているけれど、ステップには紐付いていないToDoです。

ロードマップ画面（Growth）

ロードマップの作成・編集・削除ができます。

できること

  • • ロードマップ作成
  • • 右スワイプで削除・Polaris設定・編集
  • • 編集画面でテンプレート設定
  • • タップで理想の状態とステップ管理
  • • 長押しで並べ替え

全てのステップが完了すると、完走タブに移動します。

パートナー画面（Growth）

自分のロードマップを誰かに見守ってもらうことができます。

できること

  • • パートナー申請
  • • 申請が承認されるとパートナー関係が成立
  • • パートナー関係の取り消し

先生や家族、友達にサポートしてもらうことで、一人でくじけそうな時も頑張れます。

ダッシュボード画面（Support）

パートナーの進捗を一覧で確認できます。

パートナー状況

  • • パートナー: パートナーの合計人数
  • • 完走: ロードマップを完走したパートナー数
  • • 途中: 現在進行中のパートナー数
  • • 7日以内活動: 過去7日間にToDoまたはステップを完了したパートナー数

実績

  • • 今月/今年/全期間で完走ロードマップ・達成ステップ・完了TODOを確認
  • • 今月/今年: 期間内の達成件数を表示
  • • 全期間: 進捗率を表示
  • • 集計基準: UTC（世界標準時）*

最近のアクティビティ

パートナーが最近完了したステップを時系列で確認

分析画面（Support）

パートナーのロードマップを多角的に分析できます。

分析対象

  • • パートナーの自作ロードマップ
  • • 自分のテンプレート
  • • 他人のテンプレート

全体指標

  • • 使用者数: 分析対象のロードマップを使用しているパートナー数
  • • 達成数: 完了したステップの合計数
  • • 完走率: ロードマップを完走したパートナーの割合（%）

ステップ別完了者数

  • • ファネル分析: 各ステップの完了者数を可視化
  • • どのステップで離脱が多いかを確認できます

ステップ別平均滞在日数

  • • 各ステップの平均所要日数を確認
  • • ボトルネックとなっているステップを特定できます
  • • 集計基準: UTC（世界標準時）*

離脱リスクTOP5

  • • 現在のステップ滞在日数が長いパートナーを表示
  • • タップするとパートナーのロードマップ画面に遷移
  • • 集計基準: UTC（世界標準時）*
ロードマップ画面（Support）

共有されたロードマップの進捗詳細を確認できます。

できること

  • • 右スワイプで未来のToDo・過去のToDo・内容参照
  • • タップでロードマップの内容を確認
パートナー画面（Support）

パートナー申請の承認とパートナー関係の管理ができます。

できること

  • • パートナー申請の承認
  • • パートナー関係の取り消し
設定画面

プロフィール

  • • ニックネーム変更
  • • アバター画像変更
  • • 下スワイプ更新（パートナー件数・プロファイル情報を再取得）

プラン

  • • 現在のプラン表示（Free / Growth / Support）
  • • タップでプラン変更モーダルを表示
  • • アップグレード・ダウングレード可能

モード

  • • Growth / Support モード切り替え（Switch形式）
  • • パートナー申請件数（タップでGrowth画面へ遷移）
  • • パートナー承認件数（タップでSupport画面へ遷移）

※他モード時はモード切り替え確認ダイアログが表示されます

テンプレート

  • • テンプレート有効化（Switch形式）
    　オンにするとテンプレートが作成できるようになります
  • • テンプレート検索
    　テンプレートIDを入力してロードマップを自動作成できます

セキュリティ

  • • メールアドレス変更
    　登録メールアドレスを変更できます

サポート

  • • 操作ガイド
    　YouTube再生リストで使い方動画を確認できます
  • • お問い合わせ
    　カテゴリー選択式のメール作成画面が開きます
  • • 利用規約
  • • プライバシーポリシー
  • • 特定商取引法に基づく表記

その他

  • • ログアウト
    　確認ダイアログが表示され、ログアウトできます

危険な操作

  • • アカウント削除
    　削除したアカウントは復元できません
    　同じメールアドレスでの再登録はできません
    　別のメールアドレスで新規登録してください
    　「delete」と入力して確認
サブメニュー

開き方

画面左上の3本線（ハンバーガーメニュー）をタップすると、サブメニューが開きます。

ホーム

  • • メインのタブ画面に戻ります
  • • Growth/Supportモードで表示される画面が切り替わります

インボックス

  • • どのロードマップにも紐づかないToDo
  • • メモやアイデアの一時保存に便利

未来のToDo

  • • 予定日が明日以降のToDo
  • • Growthモードでのみ表示
  • • パートナーのロードマップからも表示可能
  • • 予定日はToDo作成・編集時に指定

過去のToDo

  • • 過去に完了したToDo
  • • Growthモードでのみ表示
  • • パートナーのロードマップからも表示可能
  • • 振り返りに便利

繰り返しToDo

  • • 定期的に繰り返すToDoの一覧
  • • Growthモードでのみ表示
  • • 作成方法: ToDoを右スワイプ→「繰り返し」
  • • 繰り返しパターン: 毎日/毎週/毎月
  • • 習慣化に便利

ToDo機能

ToDoフォーム下部のアイコンバーで、各機能の表示/非表示を切り替えられます。

タイトル

ToDoのタイトルを入力します。必須項目です。

メモ

ToDoに関する詳細なメモを追加できます。アイコンをタップでメモ欄の表示/非表示を切り替えます。

URL

関連するWebページのURLを添付できます。参考リンクや関連資料へのリンクに便利です。

予定日

ToDoを実行する予定日を設定します。カレンダーから日付を選択できます。

期日

ToDoの締め切り日を設定します。期日を過ぎると視覚的に強調表示されます。

ロードマップ

ToDoを特定のロードマップに紐づけます。ToDo画面から呼び出した時のみ表示されます。

ステップ

ToDoをロードマップ内の特定のステップに紐づけます。ロードマップ選択後に表示されます。

テンプレート機能

テンプレート有効化

設定画面でテンプレートを有効化すると、Growthモードのロードマップ画面でテンプレート機能が使えるようになります。

テンプレート作成

ロードマップ編集画面でテンプレート有効化ボタンをタップすると、ロードマップをテンプレートとして公開できます。

注意:
• テンプレート作成後は更新できません
• ステップに紐づいているToDoも共有されます

テンプレートの検索

テンプレートIDを入力すると、そのテンプレートをもとにロードマップとToDoを自動で作成できます。

自動パートナー申請

テンプレートを使用すると、テンプレート作成者に自動でパートナー申請が送られます。

ネットワーク接続

オンライン環境での利用を推奨

Polarisは、安定したネット環境でのご利用を推奨しています。

データの同期について

  • • ロードマップ・パートナー・設定
    → データベースへの保存は同期処理を行なっています
  • • ToDo
    → データベースへの保存は非同期処理を行なっています

オフラインやネット環境が不安定の動作

  • • ToDoの作成・編集は可能です
  • • ただし、データベースへの保存はされない可能性があります
  • • ネット復帰後、スマホのデータがデータベースのデータで上書きされる可能性があります

今後の対応

オフライン対応の改善については、ユーザーの皆様からのご要望を参考に検討します。

トラブルシューティング

データの不整合が発生した場合

稀に以下のような現象が発生する場合があります:
• 削除したロードマップの情報が残っている
• エラーが頻繁に表示される
• 画面の表示が正しくない

対処法

  1. 設定画面を開く
  2. 「ログアウト」をタップ
  3. 再度ログイン

これにより、データベースの最新データと同期され、正常な状態に戻ります。

注意: オンライン環境での操作を推奨します。

アカウント削除

設定画面の「危険な操作」セクションから、アカウントを削除できます。

削除されるデータ

  • • ロードマップ
  • • TODO
  • • パートナー情報

注意事項

  • • この操作は取り消せません
  • • 削除したアカウントは復元できません
  • • 同じメールアドレスでの再登録はできません
  • • 別のメールアドレスで新規登録してください
データ制限

Polarisでは、快適に利用いただくため各データの取得上限を1000件に設定しています。

対象データ

  • • ToDo（種類ごとに最大1000件）
  • • ロードマップ（最大1000件）
  • • パートナー（最大1000件）
